小作品展厅

这里不是简单的链接集合，而是一组可以单独部署、也可以作为合集浏览的静态网页作品。 每个页面都有自己的独立定位：一个偏情绪互动，一个偏东方构图玩具，一个偏复古媒介展馆。

Interactive Poetry Object

记忆花园

点击夜空，种下一句会发光的话。每一句都会长成一株微光植物，适合单独作为情绪类艺术页发布。

  • 支持随机落种、手动落种、流星雨
  • 支持复制句子、下载画面
  • 适合作为独立沉浸页

Visual Toy / Calm Composition

禅石排布

在电子白砂上拖动石头，让自动生成的耙纹围绕它们流动。更像可互动海报，而不是传统小游戏。

  • 支持拖拽排布、双击落石
  • 支持构图状态统计、空格隐藏 UI
  • 支持导出当前画面
